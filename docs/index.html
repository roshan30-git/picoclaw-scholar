Core Modules

Essential components for your academic stack. Built for performance and reliability.

picture_as_pdf v1.2

PDF Indexing Engine

Vector-based ingestion of syllabus documents. Converts unstructured PDF data into queryable embeddings instantly.

quiz BETA

Daily Quiz Generator

Automated spaced repetition system (SRS). Generates questions based on recent weak points in your chat history.

memory

Context Memory

Long-term persistence layer. The model "remembers" your self-reflections and adapts its difficulty curve.

calendar_today

Exam Scheduler

Cron-like scheduling for your exam preparation. Integrates with standard calendar formats (.ics).

Mermaid Diagrams

Text-to-Diagram rendering. Request complex flowcharts or architecture diagrams directly in the terminal output.

PYQ Predictor

Statistical analysis of Previous Year Questions to predict high-probability topics for upcoming exams.
