What It Does.

PDF Indexing

Send any PDF to the bot. It extracts text and stores it in a local knowledge base.

AI Quizzes

Ask for a quiz and get 3–5 MCQs drawn from your own notes via Gemini.

Diagram Viewer

If the AI generates a Mermaid diagram, a local HTTP link is sent so you can view it directly with full pan/zoom.

Self-Reflection

The bot periodically reflects on past sessions to improve its summaries and study recommendations.

Calendar Engine

Keeps track of your semester dates and exams to intuitively adjust its "soul" (Drill Sergeant vs. Peer).

Report Generation

Generates extremely detailed progress reports of your study sessions instantly upon request.

Native Tool Calling

Uses Gemini's native function calling to autonomously decide when to generate quizzes or index new documents.

Get Started in 3 Minutes.

Local execution on Termux. Zero cloud dependencies required.

01

Deploy Binary

$ pkg install studyclaw
02

Authenticate App

$ studyclaw auth --qr
03

Execute Runtime

$ studyclaw run
