Features

Everything you need to ace your exams

Built for GTU, Mumbai University, and other Indian engineering students who want a smarter way to study.

PDF Auto-Indexing

Send any PDF or lecture note. StudyClaw extracts and stores the content in a local SQLite knowledge base, ready to be searched and quizzed from.

Daily AI Quizzes

Receive 3–5 MCQ questions drawn from your own uploaded notes every day — automatically. The bot tracks your weak spots and quizzes you on them more.

Self-Reflection Memory

When you correct the AI ("That's wrong"), it logs the mistake to a memory file and never repeats it. The AI improves every day as you use it.

GTU Exam Calendar

StudyClaw knows your GTU semester dates, mid-sems, and major Indian holidays. It automatically shifts into "Exam Mode" as key dates approach.

PYQ Pattern Predictor

Analyzes 5 years of past papers for your subject. Predicts the 5 most likely topics with probability scores — so you study what actually matters.

Live Diagram Viewer

Ask for any diagram (OSI Model, ER Diagrams, Circuit schematics). The AI generates a Mermaid chart and sends you a live viewer link to your browser.

Setup

Get running in 3 steps

No cloud accounts, no servers, no subscriptions. Just your Android phone.

1

Install Termux

Download Termux from F-Droid. Install Go and Git in your phone's terminal environment.

pkg install golang git
2

Clone & Configure

Clone the repo, set your free Gemini API key, and run the bot in a single command.

export GEMINI_API_KEY="AIzaSy..."
go run cmd/main.go
3

Scan & Study

A QR code appears. Scan it with WhatsApp → Linked Devices. Send "Hi" to yourself and start studying.

Send: "Quiz me on Thermodynamics"
